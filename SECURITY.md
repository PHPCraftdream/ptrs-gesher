# Security Policy

## Reporting a vulnerability

Please report security issues **privately** via [GitHub Security
Advisories](https://github.com/PHPCraftdream/ptrs-gesher/security/advisories/new)
on this repository. **Do not file public issues for security bugs.**

## Response expectations

This project is in pre-1.0 development. We offer **best-effort** response
with no formal SLA guarantees while the version is `0.x`.
